# SHL Assessment Recommendation Engine - Design

## Design Philosophy

- **Modern Professional Aesthetic**: Clean, sophisticated interface that conveys trust and expertise in talent assessment
- **Corporate SaaS Style**: Professional color palette with subtle gradients and modern typography
- **User-Centric Design**: Intuitive workflow that guides users from job description input to assessment recommendations
- **Data-Driven Visualization**: Clear presentation of assessment matches with confidence scores and relevant metrics

## Color Palette

- **Primary**: Deep Navy (#1e3a8a) - Professional, trustworthy
- **Secondary**: Warm Gray (#6b7280) - Neutral, sophisticated
- **Accent**: Soft Blue (#3b82f6) - Modern, tech-forward
- **Success**: Emerald (#10b981) - Positive results
- **Background**: Off-white (#fafafa) - Clean, readable

## Typography

- **Display Font**: Inter Bold - Modern, professional headings
- **Body Font**: Inter Regular - Clean, readable content
- **Monospace**: JetBrains Mono - Technical elements, code

## Visual Effects & Styling

- **Subtle Animations**: Smooth transitions using Anime.js
- **Interactive Elements**: Hover effects on cards and buttons
- **Loading States**: Professional loading indicators during API calls
- **Data Visualization**: ECharts.js for recommendation confidence scores
- **Background**: Clean gradient with subtle geometric patterns

## Layout Structure

1. **Header**: Navigation with logo and menu items
2. **Hero Section**: Main input area for job descriptions
3. **Results Section**: Assessment recommendations with detailed information
4. **Features Section**: Explanation of the recommendation engine capabilities
5. **Footer**: Copyright and links

## Interactive Components

- **Smart Input Field**: Multi-line text area with character count
- **Recommendation Cards**: Expandable cards showing assessment details
- **Filter Controls**: Test type filters (Knowledge/Skill vs Personality/Behavior)
- **Confidence Meter**: Visual representation of match quality
- **Export Options**: CSV download functionality
