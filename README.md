## Usage

### Getting Started
1. Open `index.html` in a modern web browser
2. Enter a job description in the text area or use sample JDs
3. Click "Analyze & Recommend" to generate assessments
4. Review recommendations with confidence scores and details
5. Export results or view analytics in the dashboard
